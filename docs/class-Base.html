Emulate http request like a real browser, using curl. Support cookies, no javascript.

Parameters

$url
Request URL in curl.
$get
[Optional] Urlencoded get string, will be appended with an prefix "?" after $url.
$post

[Optional] Urlencoded post string, or be an array ["assoc" => array(), "files" => <file info>]
<file info> is also an associate array as
"<name>" => "<path>"
or
"<name>" => ["filename" => "<random filename(default)>", "type" => "<application/octet-stream(default)>", "data" => <Binary Data>].
Both sub array in this field are optional. Invalid input will be ignored without any promot.

$options
[Optional] Custom curl options ARRAY.
$resetCookie
[Optional] This will only clear cookies for the request $url.

Returns

Mixed
Content from remote server, or FALSE on error.

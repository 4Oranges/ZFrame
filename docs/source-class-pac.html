28:  * For example, the first three arguments passed into <i>setTemplateArguments()</i> will replace /&#42;%arg0%&#42;/, /&#42;%arg1%&#42;/, /&#42;%arg2%&#42;/ in order.<br>

30:  * 
31:  *
32:  * @author master
33:  */
